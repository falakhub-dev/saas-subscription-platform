# SaaS Subscription Platform

A full-stack Software-as-a-Service (SaaS) platform that allows users to subscribe to plans and manage their subscriptions with secure authentication and payment integration.

## 📌 Project Overview

This platform provides a subscription-based service model where users can create accounts, select subscription plans, and make secure payments using Stripe.

## 🚀 Features

* User authentication (Login / Signup)
* Subscription plan management
* Secure payment integration with Stripe
* Dashboard for managing subscriptions
* Responsive UI

## 🛠️ Technologies Used

Frontend:

* React.js
* Tailwind CSS

Backend:

* Node.js
* Express.js

Database:

* MongoDB

Payment Gateway:

* Stripe API
